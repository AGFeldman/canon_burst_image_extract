Extract individual CR3 images from a Canon CR3 burst/roll file. Seems to work with Canon R7 files. Extracted images work with DxO PhotoLab 8, FastRawViewer, etc., but DPP itself has trouble rendering them (maybe I just need to make the embedded jpegs match the expected size?). Unknown whether this works with R6ii or R8 files.
